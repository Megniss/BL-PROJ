# Plan: Ratings Backend

**Phase:** 1 — Fixes + Backend Foundation
**Requirements:** REQ-001 (backend), REQ-002 (backend), REQ-004
**Depends on:** owner-id-migration
**Parallel:** no

## Goal

`POST /api/ratings` is live and fully tested — accepts a valid rating from the receiver of a book in a completed swap, rejects duplicates with 422, rejects non-receivers with 403, and rejects non-accepted swaps. Browse and user profile queries include average rating data.

## Context

This plan depends on `owner-id-migration` being complete because the `RatingController::store()` authorization logic needs `swap.owner_id` to determine who received which book. After a swap is accepted:
- The requester (`swap.requester_id`) received `swap.wanted_book_id`
- The owner (`swap.owner_id`) received `swap.offered_book_id`

The unique constraint is `(swap_request_id, book_id)` — NOT `(rater_id, book_id)` — because the business rule is "one rating per book per swap". A user who receives the same book via two different swaps should be able to rate it twice.

SQLite does not enforce foreign keys by default. `PRAGMA foreign_keys = ON` must be added to `AppServiceProvider::boot()`.

Existing models: `Book` (`app/Models/Book.php`) has no `ratings` relationship yet. `SwapRequest` (`app/Models/SwapRequest.php`) has `requester`, `offeredBook`, `wantedBook` relationships and (after owner-id-migration) `owner`.

### Create ratings migration, model, and relationships

- [ ] Add `DB::statement('PRAGMA foreign_keys = ON;')` to `app/Providers/AppServiceProvider.php` inside `boot()`, before the existing `ResetPassword::createUrlUsing(...)` call. Wrap in a `if (DB::connection()->getDriverName() === 'sqlite')` check so it does not break on MySQL
- [ ] Create migration: `php artisan make:migration create_ratings_table`
- [ ] In the migration `up()`:
  ```
  Schema::create('ratings', function (Blueprint $table) {
      $table->id();
      $table->foreignId('swap_request_id')->constrained('swap_requests')->cascadeOnDelete();
      $table->foreignId('book_id')->constrained('books')->cascadeOnDelete();
      $table->foreignId('rater_id')->constrained('users')->cascadeOnDelete();
      $table->unsignedTinyInteger('stars');
      $table->text('review')->nullable();
      $table->timestamps();
      $table->unique(['swap_request_id', 'book_id']);
  });
  ```
- [ ] Create `app/Models/Rating.php` with `#[Fillable(['swap_request_id', 'book_id', 'rater_id', 'stars', 'review'])]`. Add three BelongsTo relationships: `swapRequest()` -> `SwapRequest::class`, `book()` -> `Book::class`, `rater()` -> `User::class, 'rater_id'`
- [ ] In `app/Models/Book.php`, add: `use Illuminate\Database\Eloquent\Relations\HasMany;` and `public function ratings(): HasMany { return $this->hasMany(Rating::class); }`
- [ ] In `app/Models/SwapRequest.php`, add: `use Illuminate\Database\Eloquent\Relations\HasMany;` and `public function ratings(): HasMany { return $this->hasMany(Rating::class); }`
- [ ] Run `php artisan migrate`

### Create RatingController with store action

- [ ] Create `app/Http/Controllers/RatingController.php`
- [ ] Implement `store(Request $request)`:
  1. Validate: `swap_request_id` required integer exists:swap_requests,id; `stars` required integer min:1 max:5; `review` nullable string max:1000
  2. Load the swap: `$swap = SwapRequest::findOrFail($validated['swap_request_id'])`
  3. Check swap status: if `$swap->status !== 'accepted'`, return 422 with message "Can only rate accepted swaps."
  4. Determine which book the auth user received:
     - If `$request->user()->id === $swap->requester_id` then `$bookId = $swap->wanted_book_id`
     - Elseif `$request->user()->id === $swap->owner_id` then `$bookId = $swap->offered_book_id`
     - Else return 403 with message "You are not a participant in this swap."
  5. Check for duplicate: `Rating::where('swap_request_id', $swap->id)->where('book_id', $bookId)->exists()` — if true, return 422 with message "You have already rated this swap."
  6. Create the rating: `Rating::create(['swap_request_id' => $swap->id, 'book_id' => $bookId, 'rater_id' => $request->user()->id, 'stars' => $validated['stars'], 'review' => $validated['review'] ?? null])`
  7. Return the rating with 201 status
- [ ] Register route in `routes/api.php` inside the `auth:sanctum` middleware group: `Route::post('/ratings', [RatingController::class, 'store']);` — add `use App\Http\Controllers\RatingController;` at the top
