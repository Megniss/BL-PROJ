home PC / work PC setup steps:

composer run setup      <- first time only (installs deps, copies .env, generates key, migrates, builds)

Then open .env and set:
MAILTRAP_API_KEY=<your token from mailtrap.io -> Email Testing -> inbox -> Show Credentials>
MAILTRAP_INBOX_ID=<number in the URL: mailtrap.io/inboxes/XXXXXX/messages>

Then clear config cache:
php artisan config:clear

Then run the dev server:
composer run dev

NOTE: .env is never committed — you must fill in Mailtrap credentials on every new PC.
NOTE: check php.ini has extension=curl enabled (was missing on home PC, had to uncomment it)
